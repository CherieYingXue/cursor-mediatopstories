1. GPCR 脱敏与信号放大

某激素通过 Gs 偶联受体激活腺苷酸环化酶。短时刺激后 cAMP 剧增；持续刺激 30 min 后，即使激素仍存在，cAMP 产生速率下降。加入 PKA 抑制剂只能部分恢复，加入 β-arrestin 缺失突变则显著恢复。下列解释最合理的是？

答案：C

持续刺激 GPCR 后常出现脱敏：GRK 磷酸化激活态受体，β-arrestin 结合并阻断 G 蛋白偶联，同时促进受体内吞。PKA 抑制剂只能部分恢复，说明 PKA 反馈磷酸化可能存在，但不是主效应。

  • A 错：脱敏常可逆，不等于不可逆变性。
  • B 错：PDE 可降解 cAMP，但 β-arrestin 缺失显著恢复说明受体层面很关键。
  • D 错：激素不直接降解 Gs；β-arrestin 的经典作用是受体脱敏/内吞与信号支架。
代谢 · 同位素示踪2 / 16

2. 13C 标记葡萄糖的去向

细胞在缺氧条件下加入 [1-13C] 葡萄糖。若糖酵解后丙酮酸主要还原为乳酸，下列关于乳酸标记位置的判断最合理的是？

答案：B

糖酵解中果糖-1,6-二磷酸裂解并经三碳糖异构化，葡萄糖 C1 和 C6 最终对应丙酮酸 C3。乳酸脱氢酶还原丙酮酸不改变碳骨架，丙酮酸 C3 成为乳酸甲基碳。

  • A 是常见陷阱：葡萄糖 C1 不对应丙酮酸羧基碳。
  • C 错：无氧乳酸发酵不经丙酮酸脱羧释放 CO2。
  • D 错：同位素标记追踪的是碳骨架，不是还原当量本身。
遗传 · 三点测交3 / 16

3. 基因顺序判断

三基因测交子代中，两类最多的是 ABC 与 abc；两类最少的是 AbC 与 aBc。若只考虑无双交换干扰之外的常规三点作图，哪一个基因位于中间？

答案：A

亲本型为 ABC / abc。双交换型与亲本型相比，只有中间基因的等位状态发生“翻转”。最少两类 AbC / aBc 与亲本相比只有 B 改变，因此 B 位于中间。

  • 三点作图核心：最大类是亲本型，最小类是双交换型。
  • 比较亲本型与双交换型，唯一不同的位点即中间基因。
  • D 错：双交换概率最低，通常数量最少。
群体遗传 · 选择4 / 16

4. 平衡多态

某位点 A/a 中，AA 相对适合度 0.8，Aa 为 1.0，aa 为 0.6。若随机交配且只考虑该位点选择，长期预期是？

答案：D

杂合子适合度最高，属于超显性/杂合优势，会维持两个等位基因的稳定平衡。若 AA 劣势 s=0.2，aa 劣势 t=0.4，则 A 的平衡频率约 t/(s+t)=0.4/0.6=2/3。

  • 纯合子都低于杂合子时，不会简单固定某一等位基因。
  • “只剩杂合子”不可能由孟德尔分离在随机交配下长期实现。
  • 这是镰刀型贫血等经典平衡选择模型的抽象形式。
发育 · 形态发生素5 / 16

5. 阈值模型

某胚胎前后轴由形态发生素 X 梯度指定。基因 h1 需高 X 才表达，h2 需中等 X，h3 在低 X 区表达。若 X 的降解速率降低而产生更平缓且整体更高的梯度，最可能的图式变化是？

答案：B

形态发生素通过浓度阈值指定细胞命运。若降解降低，远端细胞也暴露于更高浓度，高阈值和中阈值表达域可向远端扩展，原低浓度区域被压缩。

  • 阈值模型的关键就是“细胞读出局部浓度”。
  • 若梯度形状改变，边界位置会移动。
  • 该题考的是定性预测，而不是具体分子名称。
植物生理 · 水势6 / 16

6. 气孔关闭与水势分量

干旱时 ABA 升高导致保卫细胞失水、气孔关闭。下列对水势变化的描述最符合机制？

答案：C

ABA 信号可引发胞质 Ca2+ 等变化，激活阴离子通道，膜去极化后促进 K+ 外流。溶质减少使保卫细胞水势相对升高，水外流，膨压下降，气孔关闭。

  • A 是气孔打开时常见离子变化方向。
  • 保卫细胞膨压高通常对应气孔张开。
  • 气孔调节涉及信号转导、离子通道和水势耦合。
植物光合 · C4/CAM7 / 16

7. 碳固定策略判别

某植物夜间气孔开放、苹果酸在液泡中积累，白天气孔关闭但 Calvin 循环仍进行。最合理的判断是？

答案：A

CAM 植物夜间开放气孔，由 PEPC 初始固定 CO2 为有机酸并储存；白天关闭气孔降低失水，有机酸脱羧供 Calvin 循环使用。

  • C4 是空间分离，典型涉及叶肉细胞与束鞘细胞。
  • CAM 是时间分离，适应干旱环境。
  • 白天气孔关闭不等于 Calvin 循环无 CO2，因为储存酸可脱羧供给。
神经生理 · 电化学梯度8 / 16

8. 抑制性突触的反转电位

某神经元静息电位 -65 mV，Cl- 反转电位 -72 mV。GABA_A 受体开放时，最直接的效应是？

答案：D

配体门控 Cl- 通道开放后，膜电位趋向 Cl- 的反转电位。由于 -72 mV 比静息电位更负，可产生超极化；即便反转电位接近静息，也可通过增加膜电导产生分流抑制。

  • 离子通道效应由反转电位与当前膜电位的关系决定。
  • GABA_A 主要是 Cl- 通道，不是 Na+ 兴奋性通道。
  • 发育早期若细胞内 Cl- 高，GABA 可能去极化，这是另一种情境。
循环生理 · Frank-Starling9 / 16

9. 静脉回流增加

短时间内静脉回流增加，心肌交感兴奋不变。根据 Frank-Starling 机制，最先预期出现的是？

答案：A

静脉回流增加提高前负荷，使舒张末期容积和心肌纤维初长度增加，在生理范围内增强横桥形成与收缩力，搏出量增加。

  • 该机制是心脏内在调节，不依赖交感兴奋变化。
  • 过度牵拉会降低效率，但正常范围内前负荷增加提高搏出量。
  • B 把舒张末期容积方向写反。
免疫 · 克隆选择10 / 16

10. 亲和力成熟

二次免疫应答中抗体平均亲和力高于初次应答。最直接的细胞与分子机制是？

答案：C

亲和力成熟发生在生发中心。B 细胞抗体可变区经历 AID 介导的体细胞高频突变，高亲和力克隆更有效获取抗原并获得 Tfh 帮助，从而被选择扩增。

  • V(D)J 重排主要产生初始受体多样性，不是二次应答亲和力升高的直接机制。
  • 类别转换改变恒定区，不直接提高抗原结合亲和力。
  • 红细胞无细胞核，不进行抗体基因重排。
进化 · 系统树11 / 16

11. 祖征与衍征

在构建系统树时，为什么共享衍征比共享祖征更能支持单系群？

答案：A

共享衍征可指示从最近共同祖先继承的新特征，因此支持该祖先及其所有后代组成的单系群。共享祖征可能广泛存在于更大的祖先类群中，不能限定较小分支。

  • 衍征也可能趋同或逆转，所以需要多性状/分子数据综合。
  • 现生生物当然可保留祖征。
  • IBO 系统学题常考“祖征不能定义内部分支”。
生态 · Lotka-Volterra12 / 16

12. 捕食者-猎物相位差

在经典 Lotka-Volterra 捕食者-猎物模型中，若猎物数量先上升，捕食者数量峰值通常滞后。最合理原因是？

答案：D

猎物密度升高提供更多资源，使捕食者出生率上升或死亡率下降，但个体发育与繁殖导致种群响应滞后，因此捕食者峰值通常落后于猎物峰值。

  • 经典模型的相位差来自相互作用项和种群增长时间延迟。
  • 真实生态系统还受庇护所、替代猎物、密度制约等影响。
  • A/C 与模型假设相反。
生物统计 · 实验设计13 / 16

13. 伪重复识别

研究者将 20 株植物放入同一个高 CO2 生长箱，另 20 株放入同一个正常 CO2 生长箱，并把每株植物当作独立重复做 t 检验。最严重的问题是？

答案：B

真正独立的实验单位是生长箱而不是箱内植物。每个处理只有一个箱，无法区分 CO2 处理效应与箱体差异，属于典型伪重复。

  • 应设置多个独立生长箱随机分配到处理组。
  • 统计检验必须匹配实验单位和随机化层级。
  • 增加同一箱内植物数量不能替代处理重复。
分子生物 · CRISPR14 / 16

14. CRISPR 敲除判读

用 Cas9 靶向某编码序列第一个外显子，测序显示多数等位基因发生 1 bp 插入。最可能的功能后果是？

答案：C

编码区非 3 的倍数插入/缺失通常造成移码，改变下游全部密码子并常产生提前终止密码子。若提前终止位置满足条件，mRNA 可能被 NMD 降解。

  • 1 bp 插入比 3 bp 插入更可能造成严重移码。
  • 是否完全敲除还需考虑剪接变体、起始密码子下游重启和蛋白残余功能。
  • Cas9 造成双链断裂，修复依赖细胞 DNA 修复途径。
动物发育 · Hox15 / 16

15. Hox 基因共线性

脊椎动物中，Hox 基因在染色体上的排列顺序常与其沿身体前后轴表达边界相关。若后部 Hox 基因在更前方异位表达，最可能导致？

答案：A

Hox 基因指定前后轴区域身份。后部 Hox 基因前移表达常使较前结构获得较后区域身份，即后部化/homeotic transformation。

  • Hox 是动物体轴模式形成的核心调控因子。
  • 共线性包括空间共线性，有时也包括时间共线性。
  • Hox 表型不是代谢器官“立即消失”，而是区域身份改变。
综合实验 · 证据边界16 / 16

16. 最强因果推断

某研究发现敲低基因 X 后，细胞迁移速度下降；同时 RNA-seq 显示肌动蛋白调控基因表达变化。下列哪一组实验最能加强“X 通过肌动蛋白网络调控迁移”的因果结论？

答案：D

强因果链需要排除脱靶、证明表型可由 X 回补救援、显示肌动蛋白网络结构或动态改变，并用下游因子救援建立通路顺序。单一相关性数据不足以证明机制。

  • 独立干预减少脱靶解释。
  • 救援实验是遗传因果推断的关键证据。
  • RNA-seq 是线索，不等同于功能机制证明。
